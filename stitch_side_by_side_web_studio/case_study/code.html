The Journey to Autonomy

Στον κόσμο των υποδομών, η πληροφορία που φτάνει καθυστερημένη είναι άχρηστη πληροφορία. Η Αττική Οδός αντιμετώπιζε μια κλασική πρόκληση ψηφιακού μετασχηματισμού: είχαν τα δεδομένα, αλλά η γέφυρα μεταξύ των δεδομένων και της δημόσιας επικοινωνίας ήταν ένας "άνθρωπος με πληκτρολόγιο".

Η προσέγγισή μας δεν ήταν απλώς να φτιάξουμε ένα script. Δημιουργήσαμε ένα "ψηφιακό νευρικό σύστημα". Χρησιμοποιώντας webhooks που ενεργοποιούνται από τα traffic sensors, το middleware μας φιλτράρει τον θόρυβο, κατηγοριοποιεί τη σοβαρότητα του συμβάντος και επιλέγει την κατάλληλη οπτικοποίηση για το Twitter.

Το αποτέλεσμα; Μια εμπειρία για τον χρήστη που νιώθει "ζωντανή". Για τον οργανισμό, η απελευθέρωση 120 ωρών εργασίας τον μήνα σήμαινε ότι η ομάδα επικοινωνίας μπορούσε επιτέλους να επικεντρωθεί σε στρατηγικές καμπάνιες οδικής ασφάλειας αντί για data entry.
